## PDF Image Integration (pdf-image)

This skill includes a bundled pdf-image module for extracting images from PDF files and converting PDF pages to high-quality images.

### Paths
- references/pdf-image/pdf_to_image.py
- references/pdf-image/requirements.txt
- references/pdf-image/README.md
- references/pdf-image/QUICKSTART.md

### Recommended Workflow
1. Get the source PDF from the user.
2. Decide whether to extract embedded images or render specific pages.
3. Run pdf_to_image.py with 300 DPI for publication quality.
4. Organize output by section and figure index.
5. Insert selected images into the article draft and cite source.

### Command Examples
python references/pdf-image/pdf_to_image.py your.pdf -e
python references/pdf-image/pdf_to_image.py your.pdf -p 1 3 5 -d 300 -o output

### Integration Rules
- Keep the original collaborative writing flow unchanged.
- Treat PDF image processing as an optional pre-step.
- Skip this module when user only needs writing support.
